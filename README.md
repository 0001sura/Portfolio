# Surafel Portfolio

A premium frontend portfolio built for a freelance frontend developer who focuses on landing pages, WordPress websites, redesigns, dashboards, and polished product-facing UI.

Live project goals:

- communicate trust quickly
- showcase frontend craft with strong visual hierarchy
- feel premium without becoming noisy
- convert visitors into freelance inquiries

## Live Link

https://surafels-portfolio.vercel.app/

## Overview

This portfolio is designed as a modern single-page marketing experience. It uses clean typography, a calm neutral palette, layered motion, and strong case-study framing to position Surafel as a frontend developer who can improve both presentation and conversion quality.

The layout includes:

- a premium hero with subtle parallax motion
- selected work with larger case-study previews
- service positioning for landing pages, WordPress, dashboards, redesigns, and custom frontend work
- a perspective section that adds personality and authority
- a conversion-focused contact flow

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Features

- responsive single-page portfolio
- scroll-based reveal and parallax motion
- calm, high-end UI system
- accessible interaction patterns and reduced-motion support
- personalized content for Surafel
- freelance-ready contact and GitHub sections

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start the development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

### 4. Preview the production build

```bash
npm run preview
```

## Customization

Update content in [`src/App.jsx`](./src/App.jsx) to personalize:

- display name
- title and positioning
- services
- project case studies
- testimonials
- contact details
- GitHub and other social links

Main styling and interaction helpers live in:

- [`src/index.css`](./src/index.css)
- [`tailwind.config.js`](./tailwind.config.js)

## Deployment

This project works well on free hosting platforms like:

- Vercel
- Netlify

Because this is a Vite app, deployment is straightforward:

- build command: `npm run build`
- output directory: `dist`
